AI Tutor

An AI-powered tutor built using RAG (Retrieval-Augmented Generation), LangGraph, ChromaDB, Groq, and Streamlit.

Features
- Upload and index PDF documents
- Persistent knowledge base using ChromaDB
- Multi-document retrieval
- Conversational memory
- LangGraph agent routing
- Web search fallback using Tavily
- Source citations
- Fast inference using Groq
  
Tech Stack
- Python
- Streamlit
- LangChain
- LangGraph
- ChromaDB
- Groq API
- Tavily Search
- Sentence Transformers
